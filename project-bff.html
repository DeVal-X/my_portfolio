Description

BFF est une marketplace où les utilisateurs peuvent louer des animaux fantastiques. Conçu comme un exercice pratique, ce projet visait à nous familiariser avec le développement d'une application web robuste en un temps limité. Parmi les principaux défis que nous avons rencontrés, la gestion des données utilisateurs, la mise en place d'un système de réservation et l'intégration d'une carte interactive ont été les plus significatifs. Réussir à concevoir et intégrer ces éléments dans le délai imparti a constitué le plus grand challenge de ce projet.

Contexte

Ce projet a été réalisé en une semaine dans le cadre de ma formation au Wagon, en collaboration avec deux autres développeurs. L'objectif était de créer un prototype fonctionnel d'une application web de type marketplace, mais avec un thème original basé sur l'univers d'Harry Potter. Ce projet représentait notre première expérience dans le développement d'une application web complète, intégrant des fonctionnalités pour des utilisateurs réels, y compris la gestion de la sécurité et le stockage des données.
