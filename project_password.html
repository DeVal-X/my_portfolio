Random password generator

A webapp designed to render a random password

password generator main view

Technologies used:

HTML, CSS, JS.

Desciption

This is a random password generator, Every time the user clicks on "generate a password" it will randomly generate four different passwords. The length of the passwords is adjustable between 4 and 20 characters and include numbers, symbols, lowercase and uppercase letters.
