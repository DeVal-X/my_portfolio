VeeKite

A website designed to write about Kitesurf

VeeKite website main page

Technologies used:

HTML, CSS.

Desciption

I love kitesurf but didn't take enough time to practice it as much as I should have and when I did, I always had questions such as "Should I go to that spot?" "Is it dangerous at this hour with the tide?" "Are there any local restrictions?".
It inspired me to create Veekite, so that anyone in my situation wouldn't have to go through that.
