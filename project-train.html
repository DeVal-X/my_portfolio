Animated toy train

Un petit train qui avance sur des rails

Page principale de l'application web.

Stack

HTML, CSS

Description

Ce projet consiste en un train dessiné et animé exclusivement en CSS, évoluant de manière fluide le long de rails virtuels. Il offre une expérience visuelle dynamique et immersive, démontrant les possibilités créatives offertes par l'animation CSS pure.

Contexte

Fort d'une expérience professionnelle préalable dans le secteur ferroviaire, je suis actuellement en transition vers le domaine de la programmation. À mes débuts dans cette transition, j'ai décidé de développer mes compétences en combinant mes connaissances du monde ferroviaire avec ma nouvelle passion pour le code.
